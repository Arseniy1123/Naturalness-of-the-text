1. Описание решаемой задачи: проверка закона Ципфа-Мальдеброта, построение графиков (ожидаемого и реального)
зависимости относительной частоты слова во входном тексте от ранга (порядкового номера) слова, их сравнение и вывод.

2. Ввод в программе: программа запрашивает у пользователя на ввод имя файла, с которым в дальнейшем будет работать.
Также пользователю нужно будет ввести два числа (программа попросит об этом):
- Первое число - сколько самых частотных словоформ нужно будет вывести.
- Второе число - сколько самых частотных лемм нужно будет вывести.

3. Вывод в программе: указанное пользователем число словоформ, указанное пользователем число лемм, два графика:
- Первый график - зависимость относительной частоты словоформ от ранга слова (реальный и ожидаемый).
- Второй график - зависимость относительной частоты лемм от ранга слова (реальный и ожидаемый).
Также программа выводит посчитанное значение естественности текста.

Реальный график - график, который строится по подсчитанным в программе частотам словоформ (в случае со вторым графиком - лемм).
Ожидаемый график - график, который строится по закону Ципфа.

4. Комментарии в программе прописаны перед каждым блоком кода, библиотеки и используемые модули прокомментированы,
графики также прокомментированы.
Используемые библиотеки и модули:
1) nltk - для работы со текстом и словами текста.
2) string - для работы с пунктуацией.
3) re - для работы с регулярными выражениями.
4) SnowBallStemmer - для поиска основы каждого слова из текста
5) pymorphy2 - для выделения начальной формы каждого слова из текста.
6) matplotlib - для построения графиков

5. Естественность текста. 
Для подсчета уровня естественности текста нам нужно:
1. Найти абсолютное значение разности реальных и ожидаемых частот, деленной на реальную частоту.
2. Просуммировать полученные абсолютные величины.
3. Разделить полученную сумму на количество слов в тексте.
4. Отнять полученную величину от единицы.
5. Умножить на 100 (т.е. перевести в проценты).
6. Проделать это и для словоформ, и для лемм.
7. Имеем посчитанные естественности для словоформ и для лемм.
8. Ответом будет среднее между значениями, полученными для словоформ и для лемм (сложить эти значения и разделить на 2).
9. Проанализировать: если полученное значение больше 50%, то это означает, что текст имеет хороший уровень естественности.
